# Ближайший город

Веб-приложение для определения ближайшего города к средней точке между несколькими городами.

## Возможности

- ✅ Добавление произвольного количества городов
- ✅ Автоматическое определение координат городов
- ✅ Вычисление ближайшего города к средней точке
- ✅ Определение города и страны
- ✅ Интерактивная черная карта с маркерами
- ✅ Автоматическое сохранение в cookies
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Темная тема с типографикой Inter

## Использование

1. Откройте приложение в браузере
2. Введите название города в поле ввода
3. Нажмите "Добавить город" или Enter
4. Добавьте еще несколько городов
5. Ближайший город определяется автоматически и отображается на карте

## Технологии

- **HTML5** - структура страницы
- **CSS3** - стилизация с темной темой
- **JavaScript (ES6+)** - логика приложения
- **Leaflet** - библиотека для интерактивных карт
- **CartoDB Dark Matter** - черная карта
- **Nominatim API** - геокодирование и reverse geocoding

## Особенности

- Данные сохраняются в cookies и восстанавливаются при следующем посещении
- Карта автоматически масштабируется для показа всех точек
- Каждый город отображается синим маркером
- Ближайший город отображается красным маркером
- Поддержка удаления отдельных городов
- Можно добавить один город несколько раз
- Полностью адаптивный дизайн для мобильных устройств
